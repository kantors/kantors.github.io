Do it with Code

Do it With Code is a compaign aimed at inspiring young girls to pursue a degree in STEM. The compaigns that currently exist focus on presenting statistics about the amount of women in STEM fields. However, this approach is unsucessful in truly instill passion in young girls. Do it With Code challenges the stereotype of what it means to be an engineer, a programmer a mathemeticain or anyone intrested in technology.

Rather than focusing on the lack on women in STEM, Do it with Code showcases some of the amazing women who are in the field.
