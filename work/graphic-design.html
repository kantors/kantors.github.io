Hierarhy Compositions

The purpose of this project was to explore how color, shape and size effect the order in which elements are read in a composition. Each poster incorporates typeography, letterform, linework, photography of a mechanical object and photography of a body part. This was a 6 week project where each week, we added another element focusing on creating compositions where different elements read first.

Concept Poster

Our final project in this course was to create a poster for either a play or a book. The goal of this project was to use our understanding of visual heirarchy to portray a complex narriatve to our viewers. I choose Fahrenheit 451 as it is one of my favorite books.
