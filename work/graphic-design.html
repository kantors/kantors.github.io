Exploring form and shape for graphic design 1.

1. HIERARCHY COMPOSITIONS

Model

Micheal Rascati

Material

Can-opener, Charcoal

The purpose of this project was to explore how color, shape and size effect the order in which elements are read in a composition. Each poster incorporates typeography, letterform, linework, photography of a mechanical object and photography of a body part. This was a 6 week project where the first iterations only contained two elements. Each week, we added another element and focused on creating compositions where different elements read first.
