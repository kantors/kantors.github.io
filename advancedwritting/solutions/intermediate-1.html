Browser Extensions

If you are really serious about online privacy there are a lot of browser extensions out there that can help protect you. Disconnect is a great free browser extension that shows you what websites are tracking and allows you to block the websites that are trying to track you.

Install Disconnect:

  • 1. Go to disconnect.me
  • 2. Scroll down the page and select Get Basic
  • 3. Click Get Disconnect
  • 4. Select Add Extension
  • 5. Select tour the interface to become acquainted with the extension
